1.2 Generate report

setwd("~/Cloud/ActiveProjects/OligoMM-report")
bookdown::serve_book(getwd())
# servr::daemon_stop(1)

This will starts a browser window pointing to http://127.0.0.1:7853 presenting the report written in R Markdown and will update as soon as there are changes on .Rmd files in the workspace.

To end bookdown server type servr::daemon_stop(1) where 1 is the ID of the server running

bookdown::render_book(getwd()) ## Data structure

data/raw
files from host machine, contains coverage, lofreq and varscan output
data/annotation
annotation files for OMM12 genomes e.g. phage location (pro-hunter, fasta files and Prokka annotation)
data/tables
tables in tsv format for e.g. import with Excel
data/annotation/gff
Prokka annotation
data/rda
processed raw data in rds format generated by 01-load-data.rmd
docs
html files generated via bookdown::serve_book(getwd())
